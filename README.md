This repository contains a beginner-friendly sentiment analysis project built as part of an introductory exploration into Natural Language Processing (NLP) and machine learning. The goal of the project is to classify movie reviews from the IMDB dataset as either positive or negative, using a classic and interpretable machine learning pipeline rather than a deep learning model, making it an accessible starting point for anyone learning how text classification works under the hood.
The dataset used is the IMDB Movie Reviews dataset, which contains 50,000 labeled reviews (25,000 for training and 25,000 for testing), evenly split between positive and negative sentiment. The raw text data first goes through a cleaning stage, where HTML tags, punctuation, and inconsistent casing are stripped out to standardize the input. Since machine learning models can't work with raw text directly, the cleaned reviews are then converted into numerical features using TF-IDF (Term Frequency-Inverse Document Frequency) vectorization, which captures how important each word is to a given review relative to the entire dataset.
A Logistic Regression model is trained on these vectorized features to learn the relationship between word patterns and sentiment labels. Despite its simplicity compared to modern transformer-based architectures, Logistic Regression performs strongly on this task and serves as a useful baseline for understanding model evaluation metrics like accuracy, precision, recall, and F1-score before moving on to more complex approaches.
This project was built end-to-end in Google Colab, covering the full machine learning workflow: data loading, preprocessing, feature extraction, model training, evaluation, and inference on custom input sentences. It's intended primarily as a learning exercise to build intuition around the NLP pipeline, with the long-term goal of applying similar (but more advanced) techniques to other projects involving transformer models and fine-tuning
